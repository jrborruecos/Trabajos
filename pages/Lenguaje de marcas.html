Lenguaje de Marcas

Definición

Los lenguajes de marcas son fundamentales en el mundo de la informática y la comunicación online. HTML y XML son ejemplos clave de estos lenguajes, utilizados para estructurar, formatear y organizar la información en la web y otros entornos digitales. La asignatura de Lenguajes de Marcas y Sistemas de Gestión de Información proporciona conocimientos esenciales sobre estos lenguajes y su aplicación práctica en el contexto del Grado Superior de Administración de Sistemas Informáticos en Red.

Objetivos de la asignatura

  • Comprender la estructura de los lenguajes de marcado.
  • Desarrollar contenido web utilizando HTML.
  • Manipular datos con XML.
  • Aplicar estilos visuales con CSS.
  • Integrar multimedia y formularios en páginas web.
  • Acceder a bases de datos utilizando lenguajes de marcas.

Trayectoria

Durante este primer curso de ASIR en la asignatura de Lenguaje De Marcas, los estudiantes comenzaron explorando conceptos básicos de XML y su estructura jerárquica. Posteriormente, se introdujeron en el desarrollo web utilizando HTML y CSS, trabajando en una práctica para crear una página web con diferentes secciones y contenido.
